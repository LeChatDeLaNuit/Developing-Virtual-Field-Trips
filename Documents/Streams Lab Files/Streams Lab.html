Exercise 1: Designing An Experiment

Task 1:

Design an experiment to conduct at the stream to determine the following information: average depth, average velocity, and average discharge.

Think about the following: + What measurements do you need? + How many of each measurement should you take? + How many locations should you use? + What calculations will you need to do? + Is there anything else you want to learn?

Remember, there are a limited number and variety of tools we are able to utilize! See the “Materials” section above to see the full list. If you need some help, click the hint below.

Experimental Design Hints
  • Measurements: We need average depth, average velocity, and average discharge. The “average” part implies that we need multiple measures of each thing. Discharge is a measure of volume over time–think about the formula for calculating the volume of a rectangular prism to help guide you in determining what might be needed.
  • Number of measurements: Depth, discharge, and velocity can change a lot across a stream. Rocky Branch Creek is pretty small, so we likely don’t want too many measurements. We definitely need multiple in order to calculate an average!
  • Number of locations: For larger streams, we want larger numbers of sampling locations. Since Rocky Branch is so small, any more than 3 locations would likely lead to us having data overlap. We need more than one to ensure our trial wasn’t a fluke.
  • What to calculate: Think of the things we are expected to find in our survey. Would you be able to directly measure any of them?

IA Design
  • Measurements: We need average depth, average velocity, and average discharge. The “average” part implies that we need multiple measures of each thing. Discharge is a measure of volume over time. Velocity is a measure of distance over time; we can pull this out of the discharge and are then left with an area. This area would be the width of the stream as well as it’s average depth.
    • Width: Use a tape measure across the stream.
    • Depth: We will find the average depth by finding the depth at several points across our trial line.
    • Average Velocity: We will time how long an object at the surface takes to travel 1 meter. We will then find the average of these velocities.
  • Number of measurements: Width we will measure once at each location. Depth and velocity will be measured 5 times at each location, then averaged.
  • Number of locations: Since Rocky Branch is so small, any more than 3 locations would likely lead to us having data overlap. We need more than one to ensure our trial wasn’t a fluke. This makes 2 a natural fit: one upstream and one downstream.
  • What to calculate: All of our averages need calculating. We also need to calculate velocities from time and distances.
