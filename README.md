# Developing Virtual Field Trips

Welcome to the repository for the project "Developing Virtual Field Trips to Improve Inclusivity and Accessibility in an Introductory Geology Lab Course", funded through USC's Center for Teaching Excellence as part of their Graduate Teaching Resource Development Grant.

The project aims to develop accessible, interactive, and inquiry-driven virtual field trips that engage a broader range of learners and increase the relevance of geology to diverse student populations. This is done by designing adaptable instructional materials that foster equitable learning outcomes. Within this repository, you will find two lesson plans for accessible virtual field trips and a comprehensive instructor guides with strategies, tips, and adaptability suggestions for modifying labs in this way. This functions as both a ready-to-use toolkit for instructors and as a model for designing accessible, student-centered learning experiences.

Within this repository, you should find two main folders as well as a few files.

## The Important Files

There are two main folders within this repository that you will interact with; these are the "Documents" and "Source Files" folders.

### docs

This will contain the files intended for you to read, though contains additionally folders that provide formatting. This will contain individual files for example lab activities, as well as the comprehensive guide that explains the process and justifications for these labs. These will predominantly be in the HTML format. ### source

This folder will contain the code used to develop each of the HTML files in the documents folder. The code is contained in Quarto (*.qmd) documents and can easily be used in RStudio or similar programs.

> Important!
> For most users: Open links above to navigate to specific files.
> For people who want something downloaded to their device, use the files in the "docs" folder.
> The "source" folder contains editable Quarto project files intended for instructors or developers who wish to modify the materials.
